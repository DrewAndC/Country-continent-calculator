# Route Planner

A Java desktop application that calculates the shortest and most fuel-efficient travel routes between countries using graph-based pathfinding algorithms.

## Features

- Calculate the shortest route between countries
- Determine fuel-efficient travel paths
- Graph-based route-finding algorithms
- Object-oriented Java architecture
- Unit testing for route validation
- Maven build system

## Technologies

- Java
- Maven (MVN Wrapper)
- Visual Studio Code
- Git
- GitHub

src/

## Getting Started

### Prerequisites

- Java JDK 17 (or your project's required version)
- Git

### Run the Application

Using the Maven Wrapper:

**Windows**
```bash
mvnw.cmd clean compile
mvnw.cmd exec:java
```

**macOS/Linux**
```bash
./mvnw clean compile
./mvnw exec:java
```

Or run the project directly from Visual Studio Code.

## Testing

Run the unit tests with:

```bash
mvnw test
```

or

```bash
./mvnw test
```
